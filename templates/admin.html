CATALOG DATA PIPELINE

Acquire raw blueprints directly from global repositories, sanitize inputs, and compile the master blueprints catalog.

Connects to external repository URLs to fetch raw patch manifest data and dumps it to a local JSON file.

2. Schema Parser (parser)

Parses raw dumps, standardizes materials/sizes, resolves duplicates, and rebuilds the active database 'blueprints.json'.
